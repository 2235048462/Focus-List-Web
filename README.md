# Focus List (专注清单)

一个简单而强大的任务管理应用，帮助你保持专注并提高效率。

## 功能特点

- 创建和管理任务
- 25分钟专注计时器
- 任务分类（今日、未来、已完成）
- 本地数据存储
- 响应式设计

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage
- Flatpickr (日期选择器)
- Font Awesome (图标)

## 本地开发

1. 克隆项目
2. 使用本地服务器运行项目（例如 Live Server）
3. 在浏览器中访问 `http://localhost:5500`
